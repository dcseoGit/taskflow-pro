# 01. 프로젝트 개요

## 프로젝트명

taskflow-pro

## 목적

팀 단위 태스크 관리 웹 애플리케이션.
할 일 생성·할당·상태 추적·마감일 관리를 하나의 인터페이스에서 처리한다.

## 핵심 기능

- 태스크 CRUD (생성 / 조회 / 수정 / 삭제)
- 태스크 상태 관리 (`todo` → `in_progress` → `done`)
- 담당자 할당 및 마감일 설정
- 프로젝트(보드) 단위 태스크 분류
- 사용자 인증 (JWT)

## 사용자 유형

| 유형 | 권한 |
|------|------|
| admin | 전체 관리, 멤버 초대 |
| member | 태스크 생성·수정, 본인 태스크 삭제 |
| viewer | 읽기 전용 |

## 범위 외 (Out of Scope)

- 실시간 알림 (WebSocket) — 추후 고려
- 파일 첨부 — 추후 고려
- 외부 캘린더 연동 — 추후 고려
